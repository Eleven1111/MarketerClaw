# 合规审查 — 美国市场 (FTC)

## FTC 核心规则

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 付费披露 | 高 | 所有 sponsored/gifted 内容必须明确标注 #ad / #sponsored / "Paid partnership"，置于内容开头显眼位置。FTC Endorsement Guides 2023 |
| 虚假证言 | 高 | 用户评价必须反映真实典型体验，不能仅展示非典型最佳效果 |
| 功效声明 | 高 | 健康/美容/减重等声明需有"competent and reliable scientific evidence"支撑 |
| 比较广告 | 中 | 可以比较竞品，但对比数据必须准确且可验证 |
| "Free" 声明 | 中 | 标注"免费"时不能隐藏必要条件或额外费用 |
| 退款政策 | 中 | 如果提到退款/保证，必须披露完整条件 |
| 环保声明 | 中 | "eco-friendly/sustainable/green" 需有具体依据，FTC Green Guides |
| 儿童隐私 | 高 | 面向 13 岁以下受众的内容须遵守 COPPA |

## 美国平台规则

| 平台 | 关键规则 |
|------|---------|
| Instagram/Meta | Branded Content Tool 必须开启；Stories 中 #ad 不能被小字/折叠隐藏 |
| TikTok | 必须使用 "Branded Content" toggle；不能在 bio 规避付费披露 |
| YouTube | 必须勾选"包含付费推广"；口播中需在前 30 秒内明确告知 |
| Amazon | 禁止虚假评价和刷单；A+ Content 中不能使用未验证的"best/top/leading"声明 |
| Google Ads | 广告文案不能使用误导性 URL 或虚假倒计时；Special Categories（住房/信贷/就业）受额外限制 |
