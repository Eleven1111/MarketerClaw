# 合规审查 — 欧盟市场

## GDPR 相关

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 数据收集 | 高 | 落地页/表单收集个人数据需有 opt-in 同意机制（不能预勾选） |
| Cookie 同意 | 高 | 落地页必须有 cookie consent banner，非必要 cookie 不能默认启用 |
| 再营销 | 中 | 使用 pixel/retargeting 需在隐私政策中披露 |
| 用户画像 | 中 | 基于用户行为的个性化广告需在隐私政策中说明 |
| 数据传输 | 高 | 向 EU 以外传输数据需有合法机制（SCCs/adequacy decision） |

## EU 广告规则

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 绿色声明 | 高 | EU Green Claims Directive：环保声明需第三方认证支撑 |
| Digital Services Act | 中 | 平台广告必须标注"广告"且披露广告主和投放逻辑 |
| 比较广告 | 中 | Directive 2006/114/EC：可以比较但必须客观可验证 |
| 特殊品类 | 高 | 酒精/烟草/赌博/药品各有独立限制，需按品类逐项检查 |
