# 敏感品类额外检查（全球通用）

| 品类 | 国内规则 | 海外通用规则 |
|------|---------|-------------|
| 食品 | 不能暗示疗效、不能替代药品 | 营养声明需符合目标市场法规（FDA/EFSA/FSA）|
| 美妆 | 不能使用医疗术语 | 不能声称"治疗"或"治愈"皮肤疾病 |
| 保健品 | 必须标注"本品不能替代药物" | FDA: "This statement has not been evaluated by the FDA" disclaimer |
| 金融 | 必须提示风险、不能承诺收益 | FCA (UK): risk warnings mandatory；SEC (US): past performance disclaimer |
| 教育 | 不能承诺效果、不能贩卖焦虑 | FTC: 就业/收入声明需有数据支撑 |
| 酒精 | 不能面向未成年人 | 各国年龄限制不同（US 21+ / UK 18+ / Japan 20+）；部分国家平台广告全面禁止 |
| 儿童产品 | 未成年人保护法相关规定 | COPPA (US) / UK Age Appropriate Design Code / EU GDPR Art. 8 |
