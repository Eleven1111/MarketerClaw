当用户提到以下任意情形时使用本技能：

- "文化洞察" / "人群洞察" / "时代情绪" / "这届年轻人在想什么"
- "什么概念现在最有共鸣" / "用什么叙事打动用户"

- "品牌应该表态吗" / "这个社会议题我们要不要碰"

## 什么是文化洞察，为什么营销离不开它

策略告诉你打哪个用户，文化洞察告诉你用什么语言打动他。

同一个产品，在"精致妈妈"语境里讲的故事，和在"躺平青年"语境里讲的故事，是完全不同的两件事。哪怕产品一样，人群年龄段重叠，但内在的文化密码不一样，说错话就是在花钱买嫌弃。

文化洞察不是"现在流行什么"，而是：

- **为什么**这群人现在会被这种表达打动
- **背后**是什么价值观的迁移在驱动
- **何时**一个文化时机正在生成（而不是已经过气）
- **边界**在哪里——什么能碰，什么会踩雷

## 工作方式

1. 确认洞察维度：
   - **人群**：年龄段、生活方式、代际（Z世代/千禧/X世代/银发）
   - **地域**：一线/新一线/下沉/海外（具体国家/市场）
   - **品类语境**：美妆/食品/科技/宠物/运动... 不同品类有不同的文化规则
   - **问题类型**：找表达领土 / 诊断当前传播 / 识别踩雷风险 / 捕捉文化时机
2. 读取 `brief.md`、`research.md`（如存在）获取品牌和人群上下文
3. 产出写入 `campaigns/{project-slug}/insight.md`

## 数据收集协议

### 工具探测

执行本技能前，检查当前环境中可用的搜索或浏览工具（如 WebSearch、WebFetch、浏览器工具、MCP 搜索服务、agent-reach 等）。有搜索工具时优先使用搜索工具获取多条结果；有浏览工具时用来读取单页详情。不限定具体工具名称，使用任何能访问外部实时数据的工具即可。

### 本技能数据清单

收集以下 4 类数据，每类执行 1-3 条搜索。根据目标市场调整搜索语言（中文市场用中文关键词，海外市场用对应语言）：

| 数据类型 | 搜索模板示例 |
|---------|------------|
| 目标平台当前热词 | `[平台] 热搜 [类目] 本周`、`[平台] trending [类目]` |
| 人群讨论话题 | `[目标人群关键词] 讨论 [平台]`、`[品类] 种草 话题` |
| 近期文化事件 | `[市场] 热点事件 近期`、`[目标市场] cultural trends 2025` |
| 情绪关键词 | `[品类] 用户痛点`、`[品类] 好评关键词 差评关键词` |

### 降级处理

如当前环境无可用工具，跳过数据收集步骤，继续正常执行。在交付卡中标注：⚠️ 本次未检索实时数据，建议人工核验关键数据点。

### 引用格式

数据融入正文时用方括号数字标注来源序号，例如：

> 近 30 天该品类搜索热度上涨 47%[1]

交付卡末尾附来源清单：

📊 数据来源
[1] 来源名称 / URL，检索日期 YYYY-MM-DD
[2] ...

如数据来自推断或无确凿来源，标注为：[推断，基于可比品类/历史趋势]，勿省略标注。

## 模块 1：时代情绪扫描

每个时代有它独特的集体情绪底色，品牌传播必须在这个底色上着墨，而不是对抗它。

### 情绪层分析框架

**表层情绪（可观察）**
用户在发什么内容、转发什么话题、用什么表情包、哪类视频在爆——这是情绪的外显。

**中层情绪（可理解）**
这些外显背后，是什么感受在驱动？焦虑？渴望被看见？想逃离？追求确定性？

**深层价值观（可把握）**
这一代人相信什么、不相信什么？什么对他们是真实的，什么是"虚伪的营销"？

```markdown
## 时代情绪地图

### 当下主流情绪
| 情绪维度 | 具体表现 | 品牌机会 | 雷区 |
|---------|---------|---------|------|
| {情绪名} | {可观察到的行为/内容信号} | {可以借势的方向} | {容易踩的误区} |

### 情绪迁移趋势
- **正在上升的情绪：** {及背后的社会驱动因素}
- **正在消退的情绪：** {以及为什么现在说它会显得"过时"}
- **被压抑/未被表达的情绪：** {潜在的表达空白}
```

## 模块 2：圈层文化解码

没有统一的"年轻人"，只有无数个有各自文化密码的圈层。

### 圈层解码维度

**身份认同轴**
这个圈层通过什么来定义"我们" vs "他们"？哪些符号/词汇/行为是圈内人才懂的？

**价值观轴**
他们推崇什么，鄙视什么？什么叫"真实"，什么叫"尬"？

**叙事偏好轴**
他们喜欢什么类型的故事结构？自嘲式？励志式？反英雄？佛系躺平？

**信任机制轴**
他们信任谁？怎么形成信任？KOL还是素人？专家背书还是用户口碑？

```markdown
## 圈层文化图谱

### 目标圈层：{圈层名称}

**核心身份认同**
- 自我标签：{他们如何描述自己}
- 外部符号：{服装/消费/内容偏好中的圈层标记}
- 边界意识：{对"圈外人"入侵的敏感点}

**文化密码**
- 高频词汇/梗：{及其语境含义}
- 流行叙事模板：{当前圈层内有共鸣的内容结构}
- 禁忌表达：{会被鄙视/嘲笑的品牌语言}

**品牌进入方式**
- ✅ 可以这样进：{具体的表达姿态}
- ❌ 不要这样进：{会让品牌显得"不懂我们"的方式}
```

### 主要圈层速览（可选用）

| 圈层 | 核心关键词 | 文化密码 | 适合品类 |
|------|----------|---------|---------|
| Z世代-精致穷 | 仪式感 / 平替 / 氛围感 | 追求体验而非所有权，高审美低预算 | 美妆/咖啡/文具/低客单轻奢 |
| 职场新人-打工人 | 摸鱼 / 班味 / i人 | 以自嘲消解压力，反鸡汤但不放弃 | 效率工具/零食/身心健康 |
| 新中产-精英焦虑 | 内卷 / 教育 / 品质生活 | 消费升级但理性精算，重视隐性优越感 | 教育/健康/家居/高端消费 |
| 银发新人类 | 养生 / 旅游 / 国潮 | 消费力强但不接受被"老人"标签 | 健康/旅游/文化/家庭 |
| 小镇青年 | 省钱 / 实用 / 娱乐 | 不被消费主义绑架，重性价比和娱乐价值 | 游戏/短视频/下沉零食/本地生活 |
| 海外华人 | 身份认同 / 怀旧 / 骄傲 | 在他乡重建文化归属感 | 国货/中式美食/文化IP |

## 模块 3：地域文化差异

同一个中文表达，在不同地区有不同的文化能量。

### 中国市场地域解码

```markdown
## 地域文化地图

| 市场 | 文化底色 | 消费心理 | 内容偏好 | 禁忌 |
|------|---------|---------|---------|------|
| 北京 | 政治文化中心，知识分子气质 | 实用+身份，讲究来头 | 严肃话题接受度高 | 过度娱乐化显"low" |
| 上海 | 国际化、精致、腔调 | 品质导向，追求sophistication | 精英生活方式 | 太土气/太土味 |
| 广深 | 务实、生意、效率 | ROI导向，不爱虚头巴脑 | 实用性强，种草靠数据 | 过度情怀/意义过载 |
| 成都重庆 | 享乐主义、慢节奏、市井烟火 | 体验导向，愿意为快乐买单 | 生活感、美食、休闲 | 太卷/太焦虑的叙事 |
| 新一线/下沉 | 性价比意识强，但渴望被"看见" | 实惠+有面子的平衡 | 接地气但有品味感 | 俯视感/精英叙事 |
```

### 国际市场文化解码（出海适用）

```markdown
## 出海市场文化解码

| 市场 | 当地文化底色 | 中国品牌进入禁忌 | 可借势的文化钩子 |
|------|-----------|----------------|---------------|
| 美国 | 个人主义/多元文化/直接表达 | 文化挪用、过度集体主义叙事 | 真实性、多元包容、个人故事 |
| 东南亚 | 家庭/宗教/面子文化 | 宗教敏感话题、颜色禁忌（各国不同） | 家庭价值观、节日、美白/美肤 |
| 日本 | 细节美学/匠人精神/含蓄表达 | 过度直白推销、夸张情绪 | 品质、工艺、季节感、克制美 |
| 中东 | 伊斯兰文化/奢侈消费/家族 | 宗教禁忌（斋月/猪/酒）、女性露肤 | 清真认证、家族荣耀、奢品定位 |
| 韩国 | 潮流文化输出/外貌导向/高标准 | 抄袭指控极其敏感 | K-Beauty联动、审美共振 |
```

## 模块 4：文化时机识别

文化趋势有自己的生命周期。入场太早没人懂，入场太晚只是跟风，踩准节奏才是高手。

### 文化趋势生命周期

```
发现期 → 扩散期 → 爆发期 → 主流期 → 疲倦期 → 反叛期
```

| 阶段 | 特征 | 品牌策略 |
|------|------|---------|
| 发现期 | 小圈子在用，主流未察觉 | 先行者优势，但受众小 |
| 扩散期 | 腰部KOL开始传播，媒体刚开始关注 | **最佳入场时机** |
| 爆发期 | 全网都在做，热搜频出 | 快速跟进，差异化表达 |
| 主流期 | 人人都在用，已是常识 | 持续巩固，准备转型 |
| 疲倦期 | 用户已感到审美疲劳 | 开始退出或创新变体 |
| 反叛期 | 出现反向叙事（如"反精致"反"内卷") | 新一轮洞察起点 |

```markdown
## 文化时机评估

| 文化趋势/话题 | 当前阶段 | 入场建议 | 表达差异化方向 |
|-------------|---------|---------|--------------|
```

## 模块 5：表达领土规划

综合以上洞察，定义品牌可以"占据"的文化表达领土。

### 表达领土三层结构

**领土核心**：品牌想代表的那一个核心价值观/情感
- 不能太宽泛（"我们关心用户"）
- 不能太产品（"我们的成分最好"）
- 要和文化情绪有真实的连接

**表达方式**：用什么语言、调性、视觉风格来表达这个领土
- 是认真严肃还是轻松自嘲
- 是宏大叙事还是日常细节
- 是强情绪驱动还是理性价值呈现

**边界守护**：哪些话题/方式进入会损害品牌，哪些应该回避

```markdown
## 品牌表达领土

### 领土核心
{一句话定义品牌想代表什么文化价值}

### 可以"住"的话题区域
| 话题/议题 | 进入姿态 | 表达方式 |
|---------|---------|---------|

### 需要回避的区域
| 话题/议题 | 风险类型 | 说明 |
|---------|---------|------|

### 标志性表达参考
- 语言风格：{词汇偏好、句式特征、语气}
- 视觉调性：{色调/构图/画面感}
- 内容格式：{适合这个领土的内容结构}
```

## 模块 6：踩雷预警

文化雷区往往不是恶意为之，而是洞察不足或跟风不当。

### 常见文化踩雷类型

| 雷区类型 | 典型案例模式 | 预防方法 |
|---------|-----------|---------|
| 文化挪用 | 借用少数民族/小众圈层符号但不尊重语境 | 有没有真实的文化连接？有没有该社群的参与？ |
| 伪共情 | 假装理解某群体但表达充满优越感 | 让真实目标用户审阅文案 |
| 跟风过晚 | 热点已过疲倦期时品牌才跟上 | 做时机评估，疲倦期内容转向 |
| 强行正能量 | 用户情绪低落时品牌说"一切都会好的" | 先共鸣情绪，再给出空间 |
| 议题站队失误 | 在争议性社会议题上表态，两边都得罪 | 做受众分析，评估立场风险 |
| 刻板印象 | 对某群体（性别/地域/职业）使用固化标签 | 多元视角审查 |

```markdown
## 踩雷风险评估

| 风险项 | 当前内容/策略中的具体表现 | 风险等级 | 修改建议 |
|--------|----------------------|---------|---------|
```

## 输出模式

### 快速文化诊断

用户说"这个内容方向有没有问题"时，输出：
- 当前表达的文化定位判断
- 1-2 个潜在踩雷点
- 改进方向
- 控制在半页以内

### 完整文化洞察报告

用户要"系统做一次文化洞察"时，输出全部 6 个模块。

### 定向洞察

用户只需要某个维度（"Z世代在乎什么" / "进入东南亚市场要注意什么" / "这个话题现在值不值得做"），只输出对应模块。
