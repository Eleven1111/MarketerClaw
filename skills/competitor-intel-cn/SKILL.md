---

## 触发条件

用户提供以下任意内容时使用本技能：
- 竞品名称 / 平台 / 近期动作（任意格式：自然语句、列表、表格、CSV/TSV 粘贴）
- 要求"竞品分析"、"我们该怎么应对"、"扫一下竞品最近在做什么"

---

## 输入归一化

接受以下输入格式，自动识别并转换为统一结构：

```
竞品名 | 平台 | 动作 | 信息角度 | 可攻击点
```

- **自然语言**：提取关键字段，缺失的标注"待补充"
- **CSV/TSV 粘贴**：按列解析，忽略表头行（若存在）
- **多条混合**：逐条处理，不合并

---

## 处理流程

### 步骤 1：去重
- 相同竞品 + 相同平台 + 相同核心动作 → 合并为一条，保留信息最全的版本
- 去重后说明保留条数和丢弃原因

### 步骤 2：分类

将每条记录归入以下类型之一（可多选，以主要动作为准）：

| 类型 | 关键词示例 |
|------|-----------|
| 价格促销 | 折扣、买赠、优惠券、秒杀、大促 |
| 达人种草 | KOL、KOC、达人、探店、测评 |
| 直播转化 | 直播间、连播、主播、带货 |
| 付费投放 | 信息流、Dou+、竞价、投流 |
| 品牌叙事 | 品牌故事、创始人、功效主张、情感营销 |
| 内容铺量 | 批量笔记、话题词、搜索覆盖、矩阵号 |

无法归类时标注"综合打法"并说明理由。

### 步骤 3：逐条分析

对每条记录输出：

```
竞品：[名称]
平台：[平台名]
动作类型：[分类]
核心动作：[具体描述]
信息角度：[对方的卖点叙事]
可攻击点：[弱点或空白]
---
市场影响：[对用户认知和品类期望的具体影响]
我方启示：[对我们的策略或内容有什么参考价值]
反制建议：[可执行的差异化动作，不少于2条]
```

### 步骤 4：全局洞察（在逐条分析后输出）

```
市场机会窗：[当前竞品集中火力的方向 → 留出的空档]
高频打法警示：[如果多家竞品都在做同一件事，说明...]
建议优先补采的信息：[还缺少什么数据会让分析更准]
```

---

## 输出原则

- 默认中文输出
- 每条反制建议必须是具体动作，不给"加强内容"此类无效建议
- 若数据稀疏（少于2条有效条目），说明限制并给出最小有用分析
- 若用户要求表格，先输出 Markdown 表格，再跟完整文字分析
- 不做市场份额推断，不给无来源的百分比

---

## 典型调用示例

```
用户输入：
竞品A 最近在小红书集中铺测评达人，主打成分更轻
竞品B 在抖音搞低价秒杀，强调大促优惠

期望输出：
→ 两条去重后的结构化分析
→ 竞品A归类"达人种草 + 品牌叙事"
→ 竞品B归类"价格促销"
→ 反制建议：避免跟进价格战，围绕竞品A"体验证据薄弱"的弱点做差异化真实体验内容
→ 市场机会窗：成分主张已成标配，情绪化场景和用户信任建设是未被充分占领的方向
```
