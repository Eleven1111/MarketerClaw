---

## 触发条件

当用户说以下任意一种时使用本技能：
- "帮我审一下这份文案 / 方案"
- "上线前风险核查"
- "品牌调性检查"
- "合规扫描"
- "有没有广告法风险"

---

## 审查方式：两轮并行

不要把品牌和合规混在一起说。先完成品牌审校，再进行合规审查，分开输出，分开结论。

---

## 第一轮：品牌审校

### 检查维度

| 维度 | 检查内容 |
|------|---------|
| 语气一致性 | 文案语气是否与品牌调性（如"专业但不端着"）保持一致 |
| 人群感知 | 表达方式是否让目标人群觉得"这是说给我的" |
| 卖点还原 | 核心卖点是否被清晰传达，还是被口号稀释 |
| CTA 品牌感 | 行动引导语是否与品牌气质匹配，是否过度促销 |
| 竞品区分度 | 表达是否与竞品明显区分，还是高度同质 |

### 输出格式

```
## 品牌审校

**结论：通过 / 需修改**

**发现（逐条列出）**
- [问题描述] → [影响说明]

**修改建议**
- 原文：「...」
  建议改为：「...」
  原因：[改动逻辑]
```

---

## 第二轮：合规审查

### 检查维度

#### 广告法高风险词汇
自动识别以下类型，出现即标注：

| 风险类型 | 示例词汇 / 模式 |
|---------|--------------|
| 绝对化用语 | 最、第一、唯一、顶级、100%、极品 |
| 功效承诺 | 立刻见效、永久改善、100%有效、保证瘦 |
| 医疗暗示 | 治疗、修复、医学级、临床验证（非注册医疗产品禁用） |
| 虚假紧迫性 | 限时3小时（但实际不限时）、最后10名 |
| 不实对比 | 比[竞品]效果好X倍（无数据支撑） |
| 误导性定价 | 原价可疑的划线价、虚假"已售X万件" |

#### 平台规则风险（中国主流平台）
- **小红书**：避免"第一"类绝对词、联系方式（手机号/微信号）植入正文
- **抖音**：避免平台外导流（直接引导加微信）、夸大功效类词汇
- **微信**：避免诱导关注/转发违规话术、禁止直接导流至竞争平台
- **微博**：注意热点借势中的不当捆绑

### 输出格式

```
## 合规审查

**结论：通过 / 需修改**

**发现（逐条列出）**
- 文本位置：「...」
  风险类型：[广告法 / 平台规则]
  风险等级：高 / 中 / 低
  风险说明：[具体说明为何有风险]
  建议替换：「...」（保留原意的更稳妥表达）
```

---

## 阻塞项（只在有真正阻塞时输出）

```
## 阻塞项（必须修改才能上线）

- [阻塞原因]：[具体内容]，[违反的法律条款或平台规则编号（如有）]
  必须替换为：「...」
```

---

## 审后建议

在品牌和合规审查完成后，给出**可上线版本**（若改动较多）：

```
## 可上线版本建议

[原始内容的修改版，保留原意，去除所有问题项]
```

---

## 输出原则

- **每条问题都要给替换文案**，不只是警告
- **风险等级分三档**：高（必须修改方可上线）/ 中（建议修改）/ 低（提示性说明）
- 若文案整体质量很高，仍然需要明确写出"品牌审校：通过 / 合规审查：通过"
- 若审查对象是完整 campaign 方案（非纯文案），品牌审校针对策略表达逻辑，合规审查针对所有对外沟通语言
- 不做"整体风险较低"此类模糊结论，每个问题必须有定位和修改建议
