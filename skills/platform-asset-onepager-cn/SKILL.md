---

## 触发条件

当用户说以下任意一种时使用本技能：
- "给我小红书 / 抖音 / 微信 的素材"
- "直接给可发的版本"
- "帮我出一版内容卡 / 脚本卡 / 执行单页"
- "把方案落到某平台上"

---

## 平台识别

优先以用户指定的平台为准。若用户未指定，根据 brief 中的主平台自动选择。  
若 brief 含多个平台，**逐平台输出独立资产**，不混用格式。

---

## 各平台输出规格

### 小红书（xiaohongshu）

```
## 小红书笔记卡

**标题**（20字内，含关键词，先抛痛点或对比）
→ 示例结构：「[人群痛点] 但 [正向结果]」或「为什么 [群体] 都在选 [产品]」

**正文框架**（500字内）
- 第一段：场景代入，用"我"或"你"视角展开真实处境
- 第二段：问题揭示，聚焦用户最在意的痛点
- 第三段：产品/方案登场，用具体细节（而非口号）说明为什么有效
- 第四段：结果呈现，最好有数字、对比或情绪回落

**标签建议**：3-5个，含品类词 + 场景词 + 人群词

**CTA**：收藏 / 评论关键词领取 / 点击主页查看

**发布节奏建议**：最佳发布时段 + 首批互动操作
```

### 抖音（douyin）

```
## 抖音脚本卡

**前3秒**（hook，决定完播率）
→ 冲突开场：「[群体] 注意！你以为的 [常见误区] 其实是...」
→ 结果开场：「就这一步，[产品] 帮我解决了 [问题]」
→ 问题开场：「为什么 [问题描述]？」

**中段（主体，控制在30-60秒）**
- 10s：展示使用前状态（情绪连接）
- 15s：展示产品/方案介入（具体动作）
- 15s：展示使用后结果（尽量可见/可测量）
- 10s：补充信任状（成分 / 数据 / 用户反馈）

**结尾（最后5秒）**
- 动作指令：评论区回复"关键词"领福利 / 点击购物车 / 关注看下期

**口播语气**：[对齐品牌调性，例如：轻松幽默 / 专业可信 / 情感共鸣]

**AB版本建议**：准备2个不同开头，A版用冲突开场，B版用结果开场，按完播率择优
```

### 微信生态（wechat）

```
## 微信长文框架

**标题**（结果导向 / 问题导向，15字内）

**开头段**（设问或场景，引发读者代入）
→ 不要直接介绍产品，先建立阅读理由

**正文四段结构**
1. 痛点深挖：让读者感到"说的就是我"
2. 解决方案：引入产品/方案，用场景而非参数描述
3. 产品证明：数据点 / 成分 / 体验细节 / 用户评价
4. 案例或场景还原：一个具体故事收口

**私域承接动作**（任选1-2个）
- 文末「回复关键词获取...」
- 「扫码进群」或「加企业微信」
- 「点击查看完整活动说明」

**注意事项**：阅读时长控制在4分钟内；图片穿插建议：每500字配1张
```

### 微博（weibo）

```
## 微博内容卡

**正文**（140字内主体 + 可展开）
→ 先给结论或热点借势切入点
→ 中段快速点出产品/观点
→ 结尾带话题词和互动引导

**话题词**：#[品类话题]# #[节点或热点]#

**配图建议**：对比图 / 数字图 / 场景图（优先方形或16:9）

**扩散动作**：@相关大V / 联动转发 / 评论区置顶
```

---

### B站（bilibili）

```
## B站内容卡

**视频定位**：[测评 / 教程 / 开箱 / 深度种草]

**标题**：含核心关键词，不超过20字，偏测评语气

**视频结构**
- 开头（30s内）：说清楚"本期测什么，结论是什么"，不绕弯子
- 中段（主体）：模块化拆解，每个模块有小标题
- 结尾：给出综合推荐结论 + 评论区引导话题

**弹幕/评论区预埋话题**：一个让观众主动参与的问题

**封面设计建议**：大字结论 + 产品图，避免花哨
```

### 私域（private domain）

```
## 私域承接方案

**入口设计**
- 公众号文章末尾 → 添加企业微信 / 进群
- 短视频评论区 → DM 关键词自动回复

**欢迎语（入群/加人后24小时内发送）**
→ 第一句：感谢 + 明确这里能得到什么
→ 第二句：给到立即可用的福利或内容
→ 第三句：引导第一次互动（回复 / 提问）

**7日培育节奏**
- Day 1：欢迎 + 福利
- Day 3：干货内容或使用教程
- Day 5：UGC 分享 / 用户故事
- Day 7：活动预告或限时优惠

**转化动作**：预约试用 / 优惠券领取 / 购买链接
```

---

## 输出原则

- 指定平台后，**只输出该平台规格**，不混用其他平台格式
- 每个资产卡结尾附上**风险提示**（如有合规隐患）
- 不给"内容要有趣"这类无操作价值的建议
- 若用户提供了品牌调性，所有文案语气必须对齐
- 若产品涉及健康、美妆功效或食品，在 CTA 前加合规注意事项
